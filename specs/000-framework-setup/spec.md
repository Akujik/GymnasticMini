## 📋 概述

基于 RuoYi-Vue-Pro 后台管理系统，搭建百适体操馆业务系统的基础框架。重点不是从零构建，而是 RuoYi 框架的获取、配置和体操馆业务模块的快速集成。

## 🎯 核心目标

- ✅ **快速启动**：基于成熟框架，减少基础架构搭建时间
- ✅ **标准化**：遵循 RuoYi 最佳实践和代码规范
- ✅ **可扩展**：为8个MVP模块提供统一的基础平台
- ✅ **开发效率**：代码生成器 + 自动化工具链

## 📚 背景信息

### 技术基础
- **RuoYi-Vue-Pro**: Java 8 + Spring Boot 2.7 + Vue 3
- **GitHub地址**: https://github.com/YunaiV/ruoyi-vue-pro.git
- **数据库**: MySQL 8.0 + Redis 7.0
- **部署**: Docker + Nginx

### 业务场景
百适体操馆运营管理系统，包含8个MVP模块：
1. 用户身份系统 (001)
2. 课程展示与预约 (002)
3. 候补与补课 (003)
4. 私教课系统 (004)
5. 支付集成 (005)
6. 钱包系统 (006)
7. 管理后台 (007)
8. 标签系统 (008)
